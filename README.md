# ArtviewAR
An augmented reality mobile application built with Unity, designed for museum and gallery visitors.

---

## About
ArtviewAR enhances the gallery experience by allowing visitors to point their device at paintings and artwork to unlock immersive AR content — bringing art to life beyond the canvas.

---

## Project Contents
This repository contains the Unity source project, including:
- `Assets/` — Scenes, scripts, AR content, and artwork assets
- `ProjectSettings/` — Unity project configuration
- `Packages/` — Package manifest and dependencies

---

## Requirements
- Unity 2022.x or later
- AR Foundation + XR Interaction Toolkit packages
- Android or iOS device with ARCore/ARKit support

---

## Run the Project
1. Clone the repository
2. Open the project in Unity Hub
3. Allow Unity to import and resolve all packages
4. Build and deploy to a supported mobile device

---

## Notes
- Point your camera at a supported artwork to trigger AR content
- Ensure your device has a working camera and AR support enabled
